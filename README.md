# spellcheck

> 

## Install

```console
go get -u github.com/gomjw/spellcheck
```

## Import

```go
import "github.com/gomjw/spellcheck"
```

## Usage

#### func  CheckWord

```go
func CheckWord(input string) bool
```
CheckWord checks if the input is in a dataset of every english word.
